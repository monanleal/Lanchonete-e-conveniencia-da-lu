# Lanchonete e conveniencia da Lu

<br/>

**Descrição:**
Este projeto é um site institucional para a Lanchonete & Conveniência da Lu, desenvolvido para facilitar o acesso ao cardápio, informações de contato e realizar pedidos de forma prática e rápida. O site oferece uma interface amigável e integração direta com o WhatsApp, permitindo que os clientes façam seus pedidos online com apenas alguns cliques.

<br/>

Acesse o site: https://lanchonete-e-conveniencia-da-lu.netlify.app/

<br/>
<br/>


**Funcionalidades:**
- Visualização do cardápio de produtos e preços
- Informações de contato e localização
- Integração com WhatsApp para pedidos rápidos
- Layout responsivo para dispositivos móveis e desktop

<br/>
<br/>


**Tecnologias Utilizadas:**
- JavaScript: Funcionalidades interativas e integração com o WhatsApp
- HTML5: Estruturação do conteúdo das páginas
- CSS3: Estilização e responsividade do layout

<br/>
<br/>

**Como funciona a integração com WhatsApp?**

Ao clicar no botão de pedido, o usuário é redirecionado para uma conversa no WhatsApp, facilitando o contato com a lanchonete para realizar o pedido.
